About This Project

Project Overview

This Stock Price Prediction system leverages state-of-the-art deep learning techniques to forecast stock closing prices. Built using TensorFlow and Flask, the application processes multiple features including historical prices, trading volume, market segments, and temporal patterns to generate accurate predictions.

The model underwent extensive hyperparameter tuning using Keras Tuner's Hyperband algorithm to optimize performance, ensuring the best possible accuracy for stock price forecasting.

Key Features

  • Deep neural network with optimized hyperparameters
  • Multi-feature analysis including price, volume, and temporal data
  • One-hot encoding for categorical variables
  • StandardScaler normalization for improved model performance
  • Early stopping to prevent overfitting
  • TensorBoard integration for training visualization
  • Responsive and modern user interface
  • Real-time prediction generation

Model Architecture

The prediction model uses a sequential neural network architecture with multiple dense layers. The optimal configuration was determined through Hyperband hyperparameter tuning, which explored various combinations of:

  • Number of hidden layers (1-5)
  • Neurons per layer (32, 64, 128, 256)
  • Dropout rates (0.1-0.4)
  • Activation functions (ReLU, Tanh)
  • Learning rates (0.01, 0.001, 0.0001)
